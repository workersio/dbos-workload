existing user-attribute predicate rungs. It would fail if one scheduler entry
path omitted `schedule_name`, if schedule filters leaked manual or other
schedule rows, if client/runtime list paths diverged, or if export/import
derived identity from the current schedule table rather than persisted workflow
state.
